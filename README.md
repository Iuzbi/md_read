# MoJian Markdown

一个面向 Windows 的 Markdown 阅读与编辑器，基于 Electron + React 构建，提供安装器、文件关联、拖拽打开、同步预览与护眼主题。

## 项目定位

MoJian Markdown 不是网页工具，而是一款桌面 `.exe` 应用。

它更偏向以下使用场景：

- 日常记录与 Markdown 写作
- 本地文档快速阅读与预览
- 双栏编辑与实时排版检查
- 通过安装器分发给普通 Windows 用户使用

## 功能特性

- 原生 Windows 桌面应用，支持生成 `.exe` 安装器
- 支持打开 `.md`、`.markdown`、`.txt`
- 支持拖拽文档到窗口中直接打开
- 支持文件关联，双击 Markdown 文档可唤起应用
- 单实例模式，重复打开文件时会交给已运行窗口处理
- 编辑区与预览区双栏布局，支持同步滚动
- 编辑区与预览区宽度可自由拖动调整
- 支持新建、打开、保存、另存为
- 支持最近打开记录
- 支持复制当前文件路径、在资源管理器中定位文件
- 提供多套护眼主题，并将主题设置集中放在设置页中

## 1.1.0 改动方向

- 将原本左侧常驻操作区调整为顶部工具区
- 优化未打开文件时的首页布局，减少杂乱信息
- 清理界面中不必要的说明性文案，让界面更简洁
- 增加最近打开、未保存状态提示、快捷键支持
- 丰富主题背景预设，强化护眼和整体美观度

## 快捷键

- `Ctrl + O`：打开文件
- `Ctrl + N`：新建草稿
- `Ctrl + S`：保存
- `Ctrl + Shift + S`：另存为
- `Esc`：关闭设置面板


## 版本发布建议

建议在 GitHub 中为每个正式版本创建对应 Release，并上传：

- 安装器 `.exe`
- 对应 `.blockmap`

推荐做法：

- `v1.0.0`：保留为首个正式可回滚版本
- `v1.0.1`：保留为修复版回滚点
- `v1.0.3`：保留为上一阶段稳定版
- `v1.1.0`：设置为新的最新正式版本

## 仓库展示说明

如果你准备把它作为 GitHub 项目展示，建议在 Release 页面上传每个版本的安装器，这样访问者可以直接下载试用，而不仅仅看到源码。
