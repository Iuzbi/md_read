# MoJian Markdown

> 一款面向 Windows 的 Markdown 阅读与编辑器，基于 Electron + React 构建，提供安装器、文件关联、拖拽打开、双栏编辑预览与护眼主题切换。

## 项目简介

MoJian Markdown 是一款真正可安装、可分发的 Windows 桌面应用。

- GitHub 仓库：[Iuzbi/MoJian_Markdown](https://github.com/Iuzbi/MoJian_Markdown)

项目当前聚焦以下使用场景：

- 本地 Markdown 文档阅读与编辑
- 双栏实时预览与排版检查
- 通过 `.exe` 安装器分发给普通 Windows 用户
- 使用文件关联、拖拽和快捷方式快速打开文档

## 界面说明

### 设计方向

- 整体风格偏简约桌面软件，而不是网页应用感
- 主操作集中在顶部工具栏，减少侧栏噪音
- 设置面板只在需要时展开，主题不常驻
- 首页空状态更简洁，突出“打开文档”和“新建草稿”

### 主要区域

| 区域 | 说明 |
|------|------|
| 顶部工具栏 | 打开、新建、保存、另存为、设置 |
| 主工作区 | 左侧编辑、右侧预览 |
| 设置抽屉 | 集中管理主题切换 |
| 最近文件区 | 快速恢复最近打开过的文档 |

## 快捷键

- `Ctrl + O`：打开文件
- `Ctrl + N`：新建草稿
- `Ctrl + S`：保存
- `Ctrl + Shift + S`：另存为
- `Esc`：关闭设置面板

## 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 桌面框架 | Electron | Windows 桌面壳层 |
| 前端界面 | React 19 | 主界面渲染 |
| 构建工具 | Vite | 前端开发与打包 |
| Markdown 渲染 | react-markdown + remark-gfm | 预览渲染 |
| 安装器 | electron-builder + NSIS | 生成安装器 |

## 本地开发

npm run dist
